history = "File created by Pierre St-Laurent (pst-laurent@vims.edu) on 12-Jan-2026 12:20:27 for Holden Harris (holden.earl.harris@gmail.com). The model results for year 1985 are defined on a grid of dimensions 336x564 corresponding to an oblique stereographic projection (600m x 600m, proj +proj=stere +lon_0=283.54 +lat_0=37.75). The longitude and latitude corresponding to the 336x564 points are stored inside the variables `longitude` and `latitude`. The variable `mask_land_sea` indicates whether a particular point of the grid corresponds to a `land` point or a `water` point. The variable `time` provides the date corresponding to the 365 daily-averaged values included in this file. The results were taken from a model simulation conducted by St-Laurent using ROMS-ECB (references: https://doi.org/10.17882/99441 , https://doi.org/10.1016/j.ecss.2025.109632 , https://doi.org/10.1029/2023MS003845 ) on the VIMS cluster `James` between 2025-08-30 and 2025-10-28 (runs 1272-1274,1276-1279,1281-1285,1288-1293,1295-1301,1303-1305,1307-1309,1311-1312,1314-1315,1317-1318,1320-1322)." ;
